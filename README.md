# 📈 Sales & Demand Forecasting using Machine Learning

## 📌 Project Overview
This project was developed as part of **Future Interns – Machine Learning Task 1 (2026)**.

The objective is to build a machine learning model that forecasts future sales using historical sales data. The model helps businesses predict future demand, enabling better inventory management, staffing, and financial planning.

---

## 🎯 Objectives

- Clean and preprocess historical sales data
- Perform feature engineering using date information
- Train a Machine Learning forecasting model
- Predict future sales
- Visualize historical and forecasted sales
- Evaluate model performance

---

## 🛠️ Technologies Used

- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

---

## 📂 Dataset

The project uses a sales dataset containing:

- Order Number
- Order Date
- Customer Name
- Ship Date
- Retail Price
- Order Quantity
- Tax
- Total Sales (USD)

---

## ⚙️ Machine Learning Workflow

1. Import Libraries
2. Load Dataset
3. Data Cleaning
4. Convert Date Columns
5. Feature Engineering
6. Aggregate Daily Sales
7. Train Linear Regression Model
8. Forecast Future Sales
9. Evaluate Model
10. Visualize Results

---

## 📊 Output

The project generates:

- Daily Sales Forecast
- Future Sales Prediction
- Sales Trend Visualization
- Model Evaluation (MAE)
